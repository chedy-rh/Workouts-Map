A small project developed as a part The Complete JavaScript Course 2025 From Zero to Expert! from Udemy, consist of a map that we can use to pin our workout positions and keep our workout history
## HOW To RUN ##
Simply use Live server.
## HOW To Use it ##
1- Click on a map location
2- Fill the Form that appears on the left side
3- hit enter
4- In case you added multiple workouts you can click it tile in the side-bar and the map will automatically move to it.
